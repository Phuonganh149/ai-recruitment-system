# Code of Conduct

## Our Commitment

We are committed to creating a respectful, inclusive, and professional environment for all contributors.

## Expected Behavior

Contributors are expected to:

- Communicate respectfully.
- Give constructive feedback.
- Respect different opinions and experiences.
- Focus on improving the project.
- Help maintain a positive collaboration environment.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment or discrimination.
- Insulting or offensive comments.
- Personal attacks.
- Disruptive or unprofessional communication.
- Sharing private information without permission.

## Enforcement

Project maintainers may take appropriate action when this Code of Conduct is violated, including warning contributors or removing inappropriate content.

## Scope

This Code of Conduct applies to all project spaces, including issues, pull requests, discussions, documentation, and communication related to the project.
