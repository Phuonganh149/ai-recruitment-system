# Contributing Guidelines

Thank you for your interest in contributing to AI Recruitment System.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your task.
3. Make changes clearly and consistently.
4. Commit your changes with a meaningful message.
5. Open a Pull Request and describe what you changed.

## Branch Naming Convention

Use clear branch names such as:

```text
feature/job-management
feature/candidate-profile
fix/login-validation
docs/update-readme
