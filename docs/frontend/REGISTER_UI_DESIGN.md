# REGISTER UI DESIGN

## Mục tiêu

Thiết kế giao diện đăng ký tài khoản cho người dùng mới, đảm bảo dễ sử dụng, trực quan và hỗ trợ nhiều loại tài khoản.

## Công việc cần thực hiện

* [x] Thiết kế Wireframe trang Register.
* [x] Thiết kế Form đăng ký.
* [x] Thiết kế giao diện Responsive.
* [x] Thiết kế Validate dữ liệu phía Client.
* [x] Thiết kế luồng kết nối với API Register.

## Các thành phần giao diện

### Form Đăng ký

* Họ và tên

* Email

* Số điện thoại

* Mật khẩu

* Xác nhận mật khẩu

* Vai trò:

  * Candidate
  * Recruiter

* Nút:

  * Đăng ký
  * Đăng nhập
